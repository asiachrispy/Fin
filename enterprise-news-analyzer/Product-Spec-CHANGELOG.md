## [1.0.0] - 2026-01-21

### 变更类型：新增

### 新增功能
- **全球新闻采集**：每天自动抓取全球主流新闻媒体（BBC、CNN、路透社、彭博社等）的最新报道，提供完整的新闻数据源
- **AI 事件影响分析**：基于企业背景信息，使用 AI 分析每条新闻事件对企业业务的影响程度（1-10 分评分）
- **Web 事件展示**：通过 Web 页面展示所有事件及其分析结果，支持筛选、搜索、查看详情
- **邮件推送**（低优先级）：每天将事件分析结果通过邮件发送给指定收件人（暂缓开发）

### AI增强功能
- **自动获取企业信息**：用户只需输入企业名称，AI 自动搜索并填充企业背景信息（行业、主要业务、主要市场、竞争对手）
- **智能推送**：AI 判断事件重要性，只推送高影响事件（7 分以上），用户可自定义推送阈值
- **事件聚类**：同一事件的多个新闻报道自动聚合成一个事件，避免重复

### 非功能性需求
- **性能要求**：单次抓取 ≤ 10 分钟，单条分析 ≤ 5 秒，首页加载 ≤ 2 秒
- **数据保留**：历史数据保留 30 天
- **用户系统**：单用户系统，无需注册登录
- **技术栈**：Python + FastAPI + Vue 3（用户指定）

### 技术栈
- **前端**：Vue 3 + Element Plus
- **后端**：Python 3.10+ + FastAPI + Celery + Redis
- **数据库**：SQLite
- **AI 服务**：OpenAI API（GPT-4），备选 Claude API、Google Gemini API
- **新闻抓取**：BeautifulSoup4 + Requests + Newspaper3k
- **邮件发送**：SMTP

### 影响范围
- **所有功能**：新项目从零开始，无影响范围

### 兼容性说明
- **向后兼容**：不适用（初始版本）

## 版本号规范

遵循语义化版本规范：`MAJOR.MINOR.PATCH`

- **MAJOR（主版本号）**：不兼容的 API 修改或重大功能变更
- **MINOR（次版本号）**：向下兼容的功能性新增
- **PATCH（修订号）**：向下兼容的问题修正

示例：
- `1.0.0` - 初始版本
- `1.1.0` - 新增功能，向后兼容
- `1.1.1` - 修复 bug，向后兼容
- `2.0.0` - 重大变更，不向后兼容
